# Minesweeper OOP Project

This repository contains a Python Minesweeper game built using **Object-Oriented Programming (OOP)** concepts. 

## Project Features
- Uses classes, objects, and methods
- GUI implemented using Tkinter
- Includes main.py, cell.py, utils.py, and settings.py
- Demonstrates encapsulation and interaction between objects

## How to Run
1. Open `main.py` in Python 3.x
2. Run the script to play the game
